# sharpening with high pass filter

1. make on layer by combining all layers (select all and cmd+e)
2. copy the layer (cmd+j) (give this a name, like sharpening)
3. menu -> Image -> adjustments -> desaturate (cmd+shift+u) (must have the entire image combined to one layer)
4. allpy the filter to the desaturated layer, menu -> filter -> other -> highpass filter 1.3 - 1.4 is a good start
5. blend this layer using "linear light", use opacity to bring down the effect to
