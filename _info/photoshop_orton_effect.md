# Orton effect, adds glow by using gaussian/blurr

1. create a copy of all layers (command+j) and combined (command+e)
2. select effect blurr guassian and make it blurry

3. change blend mode to screen

4. add a curve adjustment layer above the orton layer, click on the clipping mask on the adjustment tool
