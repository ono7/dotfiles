# tips https://www.youtube.com/watch?v=2fxoM45Swww

In this video, nature and wildlife photographer Simon d'Entremont shares five hidden tips and tricks within Adobe Lightroom Classic to help users improve their post-processing workflow.
The five Lightroom tips include:

1. Adjusting Midtones (0:53 - 3:57): Instead of using the blunt histogram tool which affects highlights and shadows, you can use the Color Grading panel. By selecting the Midtones tab and using the Luminance slider, you can adjust midtones delicately without clipping your whites or blacks.

2. Straightening Crooked Buildings (3:59 - 6:22): Use the Transform tool in Guided mode to fix perspective distortion common with wide-angle lenses. By drawing two lines along the edges of a building, Lightroom will automatically straighten the structure.

3. Removing Reflections (7:30 - 9:23): Within the Remove tool under the Distraction Removal section, you can select Reflections. This AI-powered tool allows you to remove or adjust the intensity of reflections in your photos using a slider.

4. Refining Saturation (9:24 - 11:42): Increasing contrast often leads to unwanted color saturation. You can fix this by using the Refine Saturation slider located directly beneath the Tone Curve panel, allowing you to add contrast without affecting the saturation levels.

5. Advanced Brush Resizing (11:43 - 14:23): You can create perfectly straight or proportionally resizing brush strokes by clicking at a starting point, holding Shift, and clicking at an end point. This is particularly useful for applying adjustments to straight objects like beams or tapering paths that shrink in the distance.
